Introduktion

På den här sidan ser du hur några av de mest avancerade AI modellerna (specifikt språkmodeller eller LLMer) klarar högskoleprovet. Utvecklingen går snabbt framåt och det kan vara svårt att bedömma hur bra dessa nya verktyg är för att lösa olika uppgifter. Att låta modellerna göra prover som till exempel högskoleprovet erbjuder ett konkret sätt att mäta modellernas kapacitet. Det är ett verktyg som både utvecklare, forskare och företag kan använda sig av för att utvärdera vad modellerna kan användas till.

I tabellen nedan kan du se hur väl olika modeller presterar på den verbala delen på högskoleprovet som kom ut hösten 2024. Modell syftar till vilken modell som använts för att skriva provet. Den totala poängen för den verbala delen på provet kan ses under poäng. Och den normerade poängen under normerad.
